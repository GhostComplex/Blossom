## 1. 凯格尔运动完整流程

### 1.1 从首页进入

**起点：** Tab 1 - 首页  
**操作：** 点击"凯格尔运动"任务卡片

### 1.2 练习页面（收缩阶段）

**页面元素：**
- 顶部导航：返回按钮 + "凯格尔运动"标题
- 阶段标签："收缩骨盆底肌"
- 圆形计时器：显示倒计时秒数（10 → 0）
- 进度信息："还剩 6 组（共 10 组）"
- 进度点：10 个圆点，已完成的变成玫瑰金色
- 操作按钮："暂停" + "结束"

**交互逻辑：**
1. 进入页面自动开始倒计时
2. 计时器从 10 秒倒数到 0
3. 每秒更新数字 + 震动反馈
4. 倒计时结束后自动切换到"放松阶段"

### 1.3 练习页面（放松阶段）

**页面变化：**
- 阶段标签变为："放松休息"
- 计时器重新显示 10 秒
- 进度点不变（当前组尚未完成）

**交互逻辑：**
1. 自动开始 10 秒放松倒计时
2. 倒计时结束后：
   - 进度点 +1（第 4 个点变成玫瑰金色）
   - 进度信息更新："还剩 5 组（共 10 组）"
   - 自动进入下一组的"收缩阶段"

### 1.4 完成练习

**触发条件：** 完成 10 组（收缩 + 放松）

**完成动画：**
1. 显示完成提示：Toast "✓ 今日凯格尔运动已完成"
2. 震动反馈（成功震动）
3. 自动返回 Tab 2 - 任务列表
4. 任务卡片状态变成"已完成" + 绿色勾选
5. 首页任务状态同步更新

### 1.5 中途暂停/结束

**暂停：**
- 点击"暂停" → 计时器停止
- 按钮变成"继续"
- 点击"继续" → 从暂停时刻继续倒计时

**结束：**
- 点击"结束" → 弹出确认弹窗
- 弹窗文案："确定要结束练习吗？当前进度不会保存"
- 确认 → 返回任务列表（未标记完成）
- 取消 → 继续练习

## 2. 拉玛泽呼吸练习完整流程

### 2.1 从首页进入

**起点：** Tab 1 - 首页  
**操作：** 点击"拉玛泽练习"任务卡片

### 2.2 选择模式页面

**页面元素：**
- 顶部导航：返回按钮 + "拉玛泽呼吸练习"标题
- 3 个模式卡片（垂直排列）

**3 种模式：**

1. **📚 学习模式**
   - 描述："查看 6 阶段呼吸法图文教程"
   - 点击 → 进入阶段列表页

2. **🫁 跟练模式**
   - 描述："跟随动画一起练习呼吸"
   - 点击 → 进入阶段选择页

3. **📖 知识卡片**
   - 描述："了解拉玛泽分娩法原理"
   - 点击 → 跳转到 Tab 4 知识页面

### 2.3 跟练模式 - 阶段选择

**页面元素：**
- 6 个阶段卡片（垂直列表）
- 每个卡片显示：阶段名称 + 适用时机

**6 个阶段：**
1. 清洁呼吸 - 宫缩前后深呼吸
2. 胸式呼吸 - 0-3cm，缓慢深呼吸 ⭐
3. 节律呼吸 - 3-8cm，随宫缩加速
4. 喘息呼吸 - 7-10cm，浅快（嘻嘻呼）
5. 吹气呼吸 - 想用力但不能用力，吹蜡烛
6. 用力呼吸 - 全开，深吸气憋住用力

**交互：** 点击任意阶段 → 进入跟练动画页面

### 2.4 跟练动画页面（第 2 阶段示例）

**页面元素：**
- 顶部导航：返回 + "第 2 阶段：胸式呼吸"
- 呼吸动画：圆形渐变动画（放大缩小循环）
- 呼吸提示："🫁 深吸气..."
- 副标题："跟随圆圈扩大吸气"
- 进度信息："还剩 4 次呼吸"
- 操作按钮："暂停" + "下一阶段"

**动画逻辑：**
1. **吸气阶段（4 秒）：**
   - 圆形从 0.8 倍放大到 1.2 倍
   - 文字："🫁 深吸气..."
   - 颜色从浅到深

2. **呼气阶段（4 秒）：**
   - 圆形从 1.2 倍缩小到 0.8 倍
   - 文字："💨 慢慢呼气..."
   - 颜色从深到浅

3. **循环：**
   - 默认 6 次呼吸为一组
   - 每次循环"还剩 X 次呼吸"倒数
   - 完成后显示"✓ 完成"提示

### 2.5 切换阶段

**点击"下一阶段"：**
- 自动进入第 3 阶段（节律呼吸）
- 动画速度变快（配合"随宫缩加速"）
- 可以通过左右滑动切换阶段

### 2.6 完成练习

**触发条件：** 完成任意阶段练习

**完成后：**
1. Toast 提示："✓ 已完成拉玛泽练习"
2. 返回任务列表
3. 任务卡片状态更新："今天 14:20 完成"
4. 首页任务状态同步

## 3. 胎动记录完整流程

### 3.1 从任务列表进入

**起点：** Tab 2 - 任务列表  
**操作：** 点击"记录一次"按钮

### 3.2 计数器页面

**页面元素：**
- 顶部导航：返回 + "记录胎动"标题
- 计数标签："胎动次数"
- 大数字显示：当前计数（初始 0）
- +1 按钮：圆形大按钮
- 操作按钮："取消" + "完成"

**交互逻辑：**
1. 每次点击 +1 按钮：
   - 数字 +1（带动画）
   - 震动反馈
   - 按钮缩放动画

2. 点击"完成"：
   - 记录当前计数 + 时间戳
   - 返回任务列表
   - 任务卡片更新："今天记录 4 次"

3. 点击"取消"：
   - 不保存
   - 返回任务列表

### 3.3 查看历史记录

**在任务列表点击任务卡片 → 进入详情页**

**详情页显示：**
```
今天记录 3 次：
• 09:30  ●●●●● (5 次)
• 14:20  ●●●●●● (6 次)
• 19:45  ●●●●●●● (7 次)

[记录一次胎动]
```

## 4. 待产包完整流程

### 4.1 勾选物品

**操作：** 点击任意物品的复选框

**交互：**
1. 复选框从空 → 绿色勾选
2. 物品名称变灰 + 删除线
3. 顶部进度实时更新
4. 进度条动画过渡

**示例：**
- 勾选前：`已准备 12 / 50 项 (24%)`
- 勾选后：`已准备 13 / 50 项 (26%)`
- 进度条从 24% 动画过渡到 26%

### 4.2 添加自定义物品

**操作：** 点击右上角 ➕ 按钮

**弹窗内容：**
- 标题："添加物品"
- 选择分类：下拉菜单（证件类/妈妈用品/宝宝用品/住院用品）
- 物品名称：文本输入框
- 数量：可选（如"x2"）
- 备注：可选
- 按钮："取消" + "添加"

**添加后：**
- 物品自动添加到对应分类
- 分类进度更新（分母 +1）
- 总进度更新

### 4.3 批量操作

**操作：** 长按任意物品 → 进入批量选择模式

**批量模式：**
- 所有复选框变成可多选
- 底部出现工具栏："批量标记为已准备" + "取消"
- 选中多个物品后点击"批量标记"
- 所有选中物品状态一次性更新

### 4.4 隐藏已准备

**操作：** 右上角开关（齿轮图标 → 设置）

**开启后：**
- 只显示未勾选的物品
- 分类进度保持显示
- 方便集中查看要买的东西

## 5. 知识文章完整流程

### 5.1 从知识首页进入

**起点：** Tab 4 - 知识  
**两种入口：**
1. 点击分类卡片（如"拉玛泽呼吸"）→ 查看该分类所有文章
2. 点击热门文章卡片 → 直接进入文章详情

### 5.2 文章详情页

**页面元素：**
- 顶部导航：返回 + 文章标题
- 配图区域：插画/图示
- 内容分段：
  - 适用时机
  - 方法（步骤列表）
  - 注意事项
- 底部操作："收藏文章" + "跟练动画"

### 5.3 收藏文章

**点击"收藏文章"：**
- 按钮变成"已收藏"（玫瑰金色）
- Toast 提示："✓ 已添加到收藏"
- 可以在知识页面顶部"我的收藏"查看

### 5.4 跳转到跟练

**点击"跟练动画"：**
- 如果是拉玛泽文章 → 跳转到 Tab 2 拉玛泽跟练动画（对应阶段）
- 如果是凯格尔文章 → 跳转到 Tab 2 凯格尔练习
- 其他文章 → 无此按钮

## 6. 跨 Tab 交互流程

### 6.1 首页快速入口

**场景：** 用户在 Tab 1 首页点击任务卡片

**流程：**
1. 点击"凯格尔运动" → 跳转到 Tab 2 并自动进入计时器
2. 点击"拉玛泽练习" → 跳转到 Tab 2 并打开拉玛泽模式选择
3. 点击"待产包" → 跳转到 Tab 3
4. 点击"分娩知识" → 跳转到 Tab 4

### 6.2 知识 → 任务

**场景：** 在 Tab 4 阅读拉玛泽文章后想练习

**流程：**
1. 点击文章底部"跟练动画"
2. 自动切换到 Tab 2
3. 自动进入拉玛泽跟练动画页面（对应阶段）

### 6.3 任务完成 → 首页同步

**场景：** 在 Tab 2 完成任务

**同步逻辑：**
1. Tab 2 任务状态变成"已完成"
2. 实时同步到 Tab 1 首页：
   - 任务卡片变成绿色勾选
   - "今日任务"进度更新："1 / 3" → "2 / 3"

## 8. 错误与边界情况

### 8.1 网络异常

**场景：** 无网络连接（App 为纯本地 App）

**处理：** 所有功能正常使用（数据存储在本地）

### 8.2 后台运行

**场景：** 凯格尔计时器运行中用户切到后台

**处理：**
1. 计时器继续运行
2. 每个阶段结束时推送本地通知
3. 回到前台时状态保持同步

### 8.3 预产期已过

**场景：** 超过预产期还未分娩

**处理：**
1. 倒计时显示："已超过预产期 2 天"
2. 颜色变成橙色（提醒）
3. 任务功能继续可用

### 8.4 待产包全部完成

**场景：** 50 项全部勾选

**处理：**
1. 进度显示："已准备 50 / 50 项 (100%)"
2. 进度条变成绿色
3. 顶部显示庆祝动画（彩纸）
4. Toast 提示："✓ 待产包准备完成！"

## 9. 完整用户旅程示例

### 场景：新用户第一天使用

**09:00 - 早上起床**
1. 收到推送通知："今日任务提醒"
2. 打开 App → Tab 1 首页
3. 看到倒计时："距离与宝宝见面 25 天"
4. 查看今日任务：3 个任务都是"待完成"

**09:30 - 做凯格尔运动**
1. 点击"凯格尔运动"任务卡片
2. 自动进入计时器页面
3. 跟随震动提示完成 10 组（共 10 分钟）
4. ✓ 完成提示，返回首页
5. 首页任务卡片变成"已完成"

**14:00 - 空闲时间学习拉玛泽**
1. 打开 App → Tab 4 知识
2. 点击"拉玛泽呼吸"分类
3. 阅读"第 2 阶段：胸式呼吸"文章
4. 点击"跟练动画" → 跳转到 Tab 2
5. 跟随动画练习 6 次呼吸
6. ✓ 完成提示，返回首页

**18:00 - 准备待产包**
1. 打开 App → Tab 3 待产包
2. 查看"妈妈用品"分类
3. 勾选今天买的 5 样东西
4. 进度从 24% 跳到 34%
5. 记下还需要买的物品

**21:00 - 睡前记录胎动**
1. 感觉到胎动
2. 打开 App → Tab 2 任务
3. 点击"记录一次胎动"
4. 点 7 次 +1 → 完成
5. 任务列表显示："今天记录 4 次"
