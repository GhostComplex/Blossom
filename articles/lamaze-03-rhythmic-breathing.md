# 第 3 阶段：节律呼吸

**分类：** 拉玛泽呼吸法  
**阅读时间：** 3 分钟

---

## 什么是节律呼吸？

节律呼吸是分娩活跃期的核心呼吸法。随着宫缩加强，呼吸节奏也要跟着加快——宫缩强时呼吸快，宫缩弱时呼吸慢，像冲浪一样"骑"在宫缩的波浪上。

## 适用时机

- 第一产程活跃期：**宫口开 3–8cm**
- 宫缩间隔缩短（2-5 分钟一次）
- 宫缩强度：中度到强度
- 胸式呼吸已经不够用的时候

## 怎么做？

1. 宫缩来临 → **清洁呼吸**开启
2. **宫缩上升期** — 呼吸逐渐加快，从慢到快
3. **宫缩高峰期** — 呼吸最快，浅而轻，用嘴呼吸
4. **宫缩下降期** — 呼吸逐渐放慢，从快到慢
5. 宫缩结束 → **清洁呼吸**收尾

### 节奏变化（像波浪）

```
慢...慢...中...快...快快快...快...中...慢...慢
 ↑                    ↑                    ↑
宫缩开始            宫缩高峰            宫缩结束
```

## 关键要点

- **跟随宫缩节奏** — 不是匀速呼吸，是变速呼吸
- **高峰期用嘴呼吸** — 轻轻的"嘻嘻嘻"或"呼呼呼"
- **保持浅呼吸** — 高峰期不需要深呼吸，浅而快就好
- **找到注视点** — 盯着一个固定的点，帮助集中注意力

## 常见问题

**Q: 呼吸太快会不会过度换气？**
A: 高峰期的快速呼吸只持续几十秒，不会过度换气。如果头晕，在宫缩间歇期做几次深呼吸恢复。

**Q: 怎么知道什么时候加快？**
A: 跟着疼痛走。痛感加强 → 呼吸加快；痛感减弱 → 呼吸放慢。你的身体会自然引导你。

## 练习建议

- 这是最需要练习的阶段，建议每天练习 5 分钟
- 让老公模拟宫缩（用手慢慢握紧你的手臂再放松），你跟着调整呼吸
- 练习时观察自己是否能在"快"和"慢"之间自然切换
