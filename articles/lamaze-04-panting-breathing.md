# 第 4 阶段：喘息呼吸（嘻嘻呼）

**分类：** 拉玛泽呼吸法  
**阅读时间：** 3 分钟

---

## 什么是喘息呼吸？

喘息呼吸，也叫"嘻嘻呼"，是分娩过渡期使用的呼吸法。这是整个分娩过程中最痛的阶段，宫缩又强又密，喘息呼吸通过快速浅呼吸帮你"渡过"最难的关口。

## 适用时机

- 第一产程过渡期：**宫口开 7–10cm**
- 宫缩最强烈、间隔最短（1-2 分钟一次）
- 持续时间最长（60-90 秒）
- 你可能感到想用力但还不能用力的阶段

## 怎么做？

1. 宫缩来临 → **清洁呼吸**开启
2. **连续浅快呼吸** — "嘻、嘻、嘻、嘻、呼——"
3. **节奏：3-4 个短"嘻" + 1 个长"呼"**
4. 循环重复，直到宫缩结束
5. 宫缩结束 → **清洁呼吸**收尾

### 节奏示意

```
嘻 嘻 嘻 呼——  嘻 嘻 嘻 呼——  嘻 嘻 嘻 呼——
（短短短 长）   （短短短 长）   （短短短 长）
```

## 关键要点

- **"嘻"是短促的** — 像在喘气，轻而快
- **"呼"是吹出的** — 像吹蜡烛，稍长一点，帮你重新调整节奏
- **用嘴呼吸** — 这个阶段全程用嘴
- **声音可以大一点** — 不需要安静，出声反而有助于释放紧张

## 为什么用"嘻嘻呼"？

- 快速浅呼吸让你没有余力去"憋气用力"（过渡期还不能用力）
- 有节奏的呼吸给大脑一个任务，分散对疼痛的注意力
- 长"呼"防止过度换气

## 注意事项

- 这是最容易过度换气的阶段，注意"呼"的时候要充分吐气
- 嘴巴可能会干，宫缩间歇期喝一小口水
- 如果头晕或手脚发麻，减慢速度或暂时改回胸式呼吸

## 练习建议

- 每天练习 2-3 分钟即可（持续太久会头晕）
- 找到自己舒服的"嘻"的个数——3 个或 4 个都行
- 可以和老公一起练，他帮你数节拍
- 练习时记住：**真正分娩时你会自然做到**，现在练的是熟悉感
