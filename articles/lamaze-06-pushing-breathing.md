# 第 6 阶段：用力呼吸

**分类：** 拉玛泽呼吸法  
**阅读时间：** 3 分钟

---

## 什么是用力呼吸？

用力呼吸是分娩的最后阶段——宫口全开，医生说"可以用力了！"这时候你要配合宫缩，用正确的方式用力，帮助宝宝通过产道。

## 适用时机

- 第二产程：**宫口全开（10cm）**
- 医生/助产士明确说"可以用力了"
- 每次宫缩来临时配合用力

## 怎么做？

1. 宫缩来临 → **深吸一口气**
2. **憋住气，向下用力** — 像排便一样，力量集中在骨盆底
3. **憋气 5-7 秒** → 快速换气 → 再憋气用力
4. 每次宫缩期间用力 **2-3 次**
5. 宫缩结束 → **清洁呼吸**恢复

### 一次宫缩的流程

```
宫缩来了 → 深吸气 → 憋气用力(5-7秒)
         → 快速换气
         → 憋气用力(5-7秒)
         → 快速换气
         → 憋气用力(5-7秒)
宫缩结束 → 清洁呼吸 → 休息
```

## 用力的正确方式

- **力量向下** — 集中在骨盆底和会阴，不是脸和脖子
- **下巴收紧** — 下巴贴胸口，背部弯曲成 C 形
- **不要憋太久** — 每次 5-7 秒，不超过 10 秒
- **宫缩间歇完全放松** — 节省体力，准备下一次

## 常见错误

❌ **脸部用力** — 涨得满脸通红，但力气没到该去的地方
❌ **憋气太久** — 超过 10 秒会减少宝宝的氧气供应
❌ **宫缩间歇还在用力** — 浪费体力，没有效果
❌ **紧张夹腿** — 腿要打开放松，给宝宝留出空间

## 关键要点

- **听从医护指令** — 他们会告诉你什么时候用力、什么时候停
- **宫缩来了才用力** — 宫缩间歇不要用力
- **每次宫缩用力 2-3 次** — 不需要一直憋气
- **相信自己的身体** — 到了这一步，你的身体知道该怎么做

## 关于"无痛分娩"

如果选择了硬膜外麻醉（无痛分娩），你可能感觉不到宫缩。这时候：
- 护士会告诉你宫缩来了
- 你可能需要在"感觉不到"的情况下用力
- 用同样的方法：深吸气 → 憋气 → 向下用力

## 练习建议

- ⚠️ **孕晚期不要真的练习用力**，可能引发早产
- 只需要**理解流程和要点**
- 可以练习"深吸气 + 憋气 5 秒"的节奏（不要向下用力）
- 练习 C 形体位：坐在床上，下巴收胸口，感受姿势
